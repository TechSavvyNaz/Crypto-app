# Flutter Firebase Authentication Template

## Introduction
This Flutter application provides a seamless user experience with integrated Firebase Authentication, featuring a splash screen, login, and sign-up functionalities. It serves as a robust template for developers looking to implement authentication flows in their Flutter projects.

## Features
- **Splash Screen**: Displays the app logo upon launch.
- **Firebase Authentication**:
  - Email and password sign-up.
  - Email and password login.
  - Password reset functionality.
- **Responsive UI**: Compatible with both Android and iOS platforms.
